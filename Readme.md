```

```

```

```

### wakatime

```
/wakatime/<username>/stats/<option>/<range>/<width-in-px>/<number-of-columns>
```

Parameters

`option`: `languages`, `editors`, `oss`
`range`: `last_7_days`, `last_30_days`, `last_6_months`, `last_year`
`width-in-pizel`: see sizing
`number-of-columns`: `1`, `2`, `3`
